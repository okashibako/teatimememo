## 同じ子要素が並ぶ状態で戻るボタンが効かない時の修正方法

<br>タイトルを中央寄せしたが、今度はヘッダー左側に追加した戻るボタンが効かなくなっていた

原因はタイトルのwidthが設定されていない、つまり子要素が並ぶ横の範囲をタイトルコンテンツが覆いつくして<br>しまっている状態にあったためだ。

この不具合は別ファイルで記載したタイトル(absoluteで設定したコンテンツを中央表示する方法.txtに記載)の<br>

セレクタに以下のようにwidthを設定することで解消することができる。

```css:header.title
header .title{
    width:120px;
    position: absolute;
    top: 0;
    right: 0;
    left: 0;
    margin: auto;
    padding: 0;}
```
